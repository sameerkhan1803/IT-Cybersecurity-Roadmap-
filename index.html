IT & Cybersecurity Roadmap

Comprehensive Professional Syllabus (Beginner to Advanced Level)

Objective

To establish a complete learning pathway that enables individuals to progress from foundational IT knowledge to advanced networking and cybersecurity expertise.

Program Structure

Detailed Modules

Module 1: Computer Fundamentals

  • Computer Basics (Hardware vs Software)
  • CPU, RAM, Storage, Motherboard
  • Operating Systems (Windows & Linux)
  • File Systems and Processes
  • Troubleshooting

Module 2: Networking Fundamentals

  • OSI Model, TCP/IP
  • IP Addressing & Subnetting
  • Protocols (HTTP, FTP, TCP/UDP)
  • Firewall, IDS, IPS, VPN

Module 3: CCNA Concepts

  • Routing, Switching
  • VLANs, STP
  • Wireless Networking
  • Automation Basics

Module 4: Cybersecurity Fundamentals

  • CIA Triad
  • Threats & Vulnerabilities
  • Types of Hackers
  • Ethical Hacking Lifecycle

Module 5: Offensive Security

  • Web VAPT
  • SQL Injection, XSS
  • Tools: Nmap, Burp Suite, Metasploit

Module 6: Defensive Security

  • SOC & Incident Response
  • SIEM & Monitoring
  • Tools: Splunk, Wazuh, Wireshark
