Modules

Module 1: Computer Fundamentals

  • Computer Basics (Hardware vs Software)
  • Input/Output Devices
  • CPU, RAM, Storage, Motherboard
  • Operating Systems (Windows & Linux Basics)
  • File Systems and Processes
  • Troubleshooting (Boot Issues, Drivers, Performance)

Module 2: Networking Fundamentals & Advanced Concepts

  • Network Types and Topologies
  • OSI Model, TCP/IP Model
  • IP Addressing (IPv4, IPv6, Subnetting)
  • Network Devices (Router, Switch, Firewall)
  • Protocols (HTTP, FTP, SMTP, TCP/UDP)
  • DNS, DHCP, NAT/PAT
  • Firewall, IDS, IPS, VPN

Module 3: CCNA Concepts

  • Ethernet, ARP, ICMP
  • VLANs, Trunking, STP
  • Static & Dynamic Routing (OSPF, RIP, EIGRP)
  • NAT, DHCP, DNS
  • Wireless Networking
  • Network Automation Basics

Module 4: Cybersecurity Core Fundamentals

  • Cybersecurity vs Information Security
  • CIA Triad
  • Risk, Threat, Vulnerability
  • Types of Hackers
  • Common Attacks (Phishing, Malware, DDoS)
  • Ethical Hacking Lifecycle
  • Security Tools Overview

Module 5: Offensive Security

  • Ethical Hacking Methodology
  • Web VAPT (OWASP Top 10)
  • SQL Injection, XSS
  • Network VAPT
  • Tools: Nmap, Burp Suite, Metasploit

Module 6: Defensive Security

  • Security Operations Center (SOC)
  • Incident Response
  • SIEM & Log Analysis
  • Malware Analysis
  • Network Security Monitoring
  • Firewall & IDS/IPS Management
