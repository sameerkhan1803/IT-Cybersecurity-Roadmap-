IT & Cybersecurity Roadmap

Build Skills. Stay Consistent. Become Industry Ready.

1. Objective

To establish a complete learning pathway that enables individuals to progress from foundational IT knowledge to advanced networking and cybersecurity expertise.

2. Target Roles

  • Desktop Support Engineer
  • IT Support Specialist
  • Network Administrator
  • Network Engineer
  • Cybersecurity Analyst

4. Detailed Modules

Module 1: Computer Fundamentals

  • Hardware, OS, Troubleshooting

Module 2: Networking

  • OSI, TCP/IP, IP Addressing, Firewall, IDS, IPS, VPN

Module 3: CCNA

  • Routing, Switching, VLANs, Wireless

Module 4: Cybersecurity

  • CIA Triad, Attacks, Ethical Hacking

Module 5: Offensive Security

  • VAPT, Web Security, Tools

Module 6: Defensive Security

  • SOC, SIEM, Malware Analysis
