IT & Cybersecurity Roadmap

Professional Learning Path (Beginner → Advanced)

Recommended Certifications

Foundation Level

  • CompTIA ITF+
  • CompTIA A+

Networking

  • CompTIA Network+
  • Cisco CCNA

Cybersecurity

  • CompTIA Security+
  • CEH (Certified Ethical Hacker)

Advanced Security

  • eJPT
  • CompTIA CySA+
  • Cisco CyberOps Associate

Objective

To establish a complete learning pathway that enables individuals to progress from foundational IT knowledge to advanced networking and cybersecurity expertise.

Detailed Modules

Module 1: Computer Fundamentals

Computer Basics, Hardware, OS, Troubleshooting

Module 2: Networking

OSI, TCP/IP, IP Addressing, Protocols, Firewall, IDS, IPS, VPN

Module 3: CCNA

Routing, Switching, VLANs, Wireless, Automation

Module 4: Cybersecurity

CIA Triad, Hackers, Attacks, Ethical Hacking

Module 5: Offensive Security

VAPT, Web Security, Network Testing, Tools

Module 6: Defensive Security

SOC, SIEM, Malware Analysis, Monitoring
